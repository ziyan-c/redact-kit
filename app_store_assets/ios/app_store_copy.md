## Description

```text
Redact Kit is a focused image privacy tool for redacting pixels and removing private metadata.

Use Redact mode to cover sensitive areas with solid pixels, then export a clean PNG or JPEG copy. Redaction is burned into the raster, so the exported image has no editable layer, hidden mask, or original pixels underneath the covered area.

Use Clean Metadata mode to create clean image copies without common private metadata such as EXIF, GPS, IPTC, XMP, thumbnails, and comments.

Designed for simple local workflows:

- Open images from Files or Photos
- Draw solid redaction boxes
- Remove metadata from images
- Choose PNG or JPEG export
- Save to Files, save to Photos, or share

Redact Kit does not require an account. Images are processed locally on your device unless you choose to save or share the exported result.
```

## App Review Notes

```text
Redact Kit is a local image privacy utility. To test Redact mode, open an image from Files or Photos, draw a redaction box, then export. To test Clean Metadata mode, switch to Metadata, select one or more images, and tap Start to save cleaned copies.

The app does not require an account and does not upload images. Photos and Files access are used only when the user chooses input images or saves exported output.
```
